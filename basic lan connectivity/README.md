4 PCs Connected to Switch
📌 Overview

Simple Star Topology using Cisco Packet Tracer where 4 PCs are connected to a switch.

🖥️ Setup
1 Switch
4 PCs
Ethernet cables
🌐 IP Address
PC0 → 192.168.1.1
PC1 → 192.168.1.2
PC2 → 192.168.1.3
PC3 → 192.168.1.4
🚀 Usage

Open .pkt file → Assign IP → Use ping to test.

✅ Result

All PCs communicate successfully.
