## 🚀 What this project does

This project is a Cisco Packet Tracer network simulation. It shows a basic network with a router, a switch, and end devices. It also includes IP addressing and ping checks that confirm the devices can reach each other.

Use it if you want to open a ready-made network file in Packet Tracer and study how a simple setup works. It is made for Windows users who want to download the file, open it, and view the network layout without building it from scratch.

## 🔌 How to use the project

After the file opens, you can view the network and inspect each device.

- Click the router to see its settings
- Click the switch to view its ports
- Click a PC or other end device to check its IP settings
- Open the command prompt in a device to view ping results
- Follow the cable lines to understand how devices connect

If you want to learn from the project, start with the IP addresses. Then check how the router and switch are linked. After that, review the ping test to see which devices can reach each other.

## 📶 Network layout

This project uses a basic LAN-style setup with routing support. The layout is simple and easy to follow:

- One router handles network traffic
- One switch connects local devices
- End devices use static IP addresses
- Devices communicate through the network
- Ping confirms the setup works

This type of layout is useful for learning the first steps of network design in Packet Tracer.
