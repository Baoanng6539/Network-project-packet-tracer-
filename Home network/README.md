🏠 Home Network using Cisco Packet Tracer
📌 Overview

This project simulates a simple home network using Cisco Packet Tracer. Multiple devices are connected through a router and switch to enable communication and internet access.

🖥️ Network Devices
Router
Switch
PCs / Laptop
Smartphone (optional)
🌐 Network Description

All devices are connected to a router through a switch.
The router manages the network and allows devices to communicate with each other.

🚀 Usage
Open the .pkt file in Cisco Packet Tracer
Check device connections
Test connectivity using ping
🎯 Outcome
Understand basic home network setup
Learn device connectivity and communication
Practice network simulation
